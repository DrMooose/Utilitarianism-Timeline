Six Ethical Philosophies

The six ethical philosophies are an important aspect in analyzing situations and the ethical actions that are behind them. Many pieces of work can ethically be separated into one of these six categories. To be able to understand each philosophy is an asset to any individual looking to find the ethics behind others actions. This project looks to try and create a site that will examine all six ethical philosophies as an interactive web experience. Each philosophy will be sectioned off into its own area of the page, with a color that corresponds to the core values of the philosophy
